YouTube がテレビになった日 — 35〜64歳を "取り戻す" 2つのフォーマット

Five Stars① 世界ニーズ抽出 / ③ ROIC最大化 更新2026-05-24 (v2) 方式マーカー位置縛りなし
⚠️ このブラウザは Web Speech API に未対応の可能性があります。Chrome / Edge を推奨します(動かない場合は下部テキスト欄に直接入力してください)。

米国のリビングで、ある数字が静かに反転した。YouTubeをテレビ画面で観る時間が、スマホで観る時間を超えたのだ。35歳から64歳、つまり地上波の黄金期にチャンネルを回し続けてきた世代からTVを視聴する最後の世代までが、リモコンを握ったままYouTubeを「夜のテレビ」として消費し始めている。Streaming MediaのYear in Review 2026も、Deloitteの2026 M&E Outlookも、揃ってこの地殻変動を年の最大トピックに据えた。スマホで切り抜きを摘み食いする若者の物語ではない。むしろ逆で、ソファに座った大人が、CMの間にチャンネルを変える代わりにサジェスト動画をタップしている、という話なのだ。

ここで多くの放送関係者が誤読する。「ではYouTubeにもっと動画を出そう」と。私は、そうではないと思う。テレビ画面で観られるYouTubeに必要なのは、本数ではなく編成だ。何時から、誰に向けて、どんな並びで、何を当てるか——地上波が半世紀かけて磨いた時間設計の技術こそ今、YouTubeに活用できると思う。切り抜きチャンネルが伸びても放送局の力にはならない。帯の感覚、特番の呼吸、改編期の緊張、視聴者を「今夜見るべきもの」へ繋げる導線の設計。これらこそ、番組を作る力とは別の、編成という地上波の部署が培ってきた筋肉であり、今使うべきものだ。

逆に言えば、この筋肉を持っている組織は世界にそう多くない。米国のネットワーク局、英国のBBC、そして日本のキー局と一部の地方局などではないか。理由は意外と単純だ。編成の腕は「少数のチャンネルが、同じ時間に大勢が観る前提で、何十年も競争しながら時間を設計し続けた」環境で育つ。多チャンネルのケーブルが普及していた国では、視聴が最初から細切れに分散し、一本の流れを編む筋肉がそもそも要らなかった。国営一極で競争のない国でも、数字を奪い合う改編期の緊張は生まれなかっただろう。広告か受信料で支えられた少数の放送局が、毎日24時間ぶんの並びを、何世代も競い合って磨き続けた——それがたまたま米・英・日だった、という話だ。だからこの3国の編成筋肉は、世界的に見ればきわめて希少な筋力なのである。テレビ画面化したYouTubeは、皮肉なことに、いま最も「編成の効く媒体」へと変貌しつつある。アルゴリズムが個人最適を担う一方で、リビングの大画面では家族や夫婦が並んで観るため、個人最適だけでは間が持たない。誰かが束ねた「今夜見るべきもの」を、視聴者は無意識に求め始めている。

時差を伴って、この波は必ず日本に来る——いや、すでに来ていると思われる。60代のスマートテレビ普及率は静かに上がり、リモコンにYouTubeボタンが載った瞬間から、地上波とYouTubeは同じ画面の中で隣り合うライバルになった。問題は、日本のシニア層を「取り戻す」側に回れるのは誰か、だ。広告単価が崩れ、制作費が削られ、若年層がそもそもテレビの前にいない構造の中で、35〜64歳という最後の優良視聴者層をYouTubeに明け渡すか、それとも自局のYouTubeチャンネルへ連れて行くか。経営判断としては、後者に挑戦する意思決定が必要だ。

ただし、地上波の番組をそのまま縦に切って上げても届かない。テレビ画面で観るYouTubeには、テレビ画面で観るYouTubeの編成アプローチがいる。米国先行事例から逆算すると、日本のシニア層を取り戻すには少なくとも2つのフォーマットが要る、というのが現時点での私の仮説だ。

1つ目は「夜帯ニュース・ループ」。20分前後の地域ニュースを1時間ごとに差し替え、夕方から深夜までソファに張り付いた視聴者がいつ点けても「今の地元」が流れている状態を作る。2つ目は「対談型・帯番組」。スタジオもセットも最小、ホスト1人とゲスト1人、毎週同じ曜日同じ時間に必ず更新する。シニア層が最も信頼するのは「いつもの時間にいつもの顔がいる」体験で、これは技術ではなく規律の問題に近い。

YouTubeがテレビになった日、勝つのは動画が上手い組織ではなく、時間を編める組織ではないだろうか。

編成会議という場がある。

縦に視聴率帯、横にターゲット属性、奥行きに尺とCM枠。この三次元の連立方程式を、毎週ホワイトボードの前で解いていた。Aターゲット(男女13-49)を何時に何分掴むか。F1とF2の重なりをどこで切るか。深夜は誰のための器か。あの会議室で起きていたことが、いまYouTubeの中で再演されている。

そう感じたのは、Streaming Media Year-in-Review 2026(取得日:2026-05-23 / 信頼度:高)を読んだときだ。米国では、テレビ画面で見られるYouTubeが、モバイルでのYouTubeを超えた。同時にYouTubeのCEOは「YouTube is the new television」と言い切った(信頼度:高)。

これは「若者がスマホで短い動画を見ている」という、よくある物語の続編ではない。35〜64歳が、テレビのリモコンでYouTubeを立ち上げ、リビングのソファに沈み込んでいる。ここから先は、編成の話だ。

1. 何が起きたか ── 米の事実と、日本の兆候

事実を順番に置く。

  • 米国のTV画面におけるYouTube視聴時間が、モバイル視聴を超えた(2026 / 出典:Streaming Media Year-in-Review 2026 / 信頼度:高)
  • 米国のインターネットユーザーの85%が、月1回以上YouTubeを見ている(同 / 信頼度:高)
  • 35〜64歳が "golden age of film and TV" のYouTube視聴を牽引している(同 / 信頼度:高)
  • SVOD全体の成長率は2026年に約5%、2030年には2%を割る見通し(同 / 信頼度:中)
  • アニメ視聴では、調査対象9カ国のうち日本だけがYouTubeを首位に据えている(同 / 信頼度:中)

ここで「アメリカは特殊だから」と言いたくなる。が、それは構造を見ていない。日本でも、スマートTVの出荷比率は高止まりし、HDMI 1番にYouTubeアプリが並ぶ家庭は珍しくない。リビング動線で「テレビをつける = YouTubeを開く」という置き換えが、静かに進んでいる(推定 / 要検証)。

【日本市場との適合度評価】 米国の構図がそのまま再現される確度:中。ただし「TV画面でYouTubeを見る35〜64歳」というセグメントは、日本では地上波視聴の慣性が強く、置換ではなく "並走" になる可能性が高い。結論:「日本では、YouTubeはテレビの代替ではなく、テレビの一画面として共存する」と置いて編成を組むのが筋がいい。

2. なぜ35〜64歳が黄金層なのか ── 可処分所得 × 可処分時間 の交点

メディアの収益は、視聴者の頭数ではなく、視聴者の財布で決まる。広告主が買っているのは「届く人数」ではなく「届いた先で動く金額」だ。ここを取り違えると、編成は永遠に若者を追って疲弊する。

35〜64歳は、可処分所得と可処分時間の積が最大になる帯だ。住宅ローンが終わりに近づき、子の手が離れ、健康投資・趣味・学び直し・旅行・資産運用に予算が動く。広告主にとって、これほど都合のいい層はない。CPM(千回表示あたり広告単価)は、若年向けインフィード広告より、シニア向け健康・金融・旅行のほうがしばしば2〜3倍高い(推定 / 業界実勢 / 信頼度:中)。

つまり、YouTubeの35〜64歳は、編成的に言えば「ゴールデン」だ。夜7時から11時の地上波で取りに行っていた、あの層が、リビングの同じソファでYouTubeを見ている。取りに行かない理由がない。

3. 地上波編成ロジックをYouTubeに転写する3つのフォーマット

ここからが本題だ。編成の三種の神器 ── 帯・特番・深夜枠 ── をそのままYouTubeに転写する。器が変わっただけで、視聴行動の本質は変わっていない。

フォーマット① 帯番組型(週次 × 固定時間 × 固定MC)

地上波の朝の情報番組、夕方のニュース、平日深夜のバラエティ。これらが視聴習慣として強いのは「同じ時間・同じ顔・同じ枠」で記憶に刻まれるからだ。

YouTubeでも同じことができる。週1〜2回、決まった曜日の決まった時間に、決まったMCが出る。動画長は15〜25分。冒頭フォーマット、コーナー構成、エンディングを固定する。チャンネル登録者の「予測可能性」を上げると、通知ON率と初動再生が跳ねる。

  • 想定CPM:¥800〜¥1,500(日本/35〜64歳/長尺ミッドロール想定 / 推定 / 信頼度:中)
  • 想定ROIC:制作費30〜50万円/本、年間60本、広告+案件で粗利率35〜45%を狙う規模感(推定 / 要検証)

地上波1番組の予算感とは桁が違う。だからこそ、ROICで見ると勝てる構造になりうる。

フォーマット② 特番型(イベント連動の高密度尺)

選挙、スポーツ国際大会、決算シーズン、大型新商品発表、地域の祭礼。地上波の「特番」は、視聴者の関心が集中する瞬間に、高密度で尺を取る装置だ。

YouTubeの特番は、生配信 + 編集アーカイブの二段構えで効く。当日:2〜4時間の生配信で同時接続を稼ぎ、スーパーチャットと案件タイアップを取る。翌週以降:30分×3〜5本に再編集してロングテールを取る。帯番組のチャンネル登録者を母集団として動員すれば、初動が安定する。

  • 想定CPM:¥1,200〜¥2,500(イベント連動の希少性プレミアム / 推定 / 信頼度:中)
  • 想定ROIC:1イベントで制作費の1.5〜3倍回収を狙う(推定 / 要検証)

フォーマット③ 深夜枠的尖り(ニッチ深掘り × ロイヤリティ醸成)

地上波の深夜は、視聴率は低いがロイヤリティが高い。熱量の濃い1万人は、薄い100万人より、広告主と物販に強い。

YouTubeのニッチチャンネルは、まさに深夜枠の構造だ。登録者1〜5万人でも、視聴完了率が高く、コメント欄が機能していれば、メンバーシップ・物販・有料コミュニティでUnit Economicsが成立する。CAC Cashback(1人あたり獲得コストの回収期間)を6〜9ヶ月に収められれば、規模は小さくても自立する。

  • 想定CPM:広告依存度を下げ、メンバーシップARPU ¥500〜¥1,500/月で組む(推定 / 信頼度:中)
  • 想定ROIC:広告フォーマットより不安定だが、当たれば粗利率50%超(推定 / 要検証)

地上波の「深夜は実験場」という思想を、YouTubeで本気で運用するチームは、まだ日本に多くない。

4. 日本の中小・地方局が今週から取れる1手

ここまでの構造を踏まえて、地方局・中小制作会社・地方発の中小企業が、今週から動ける手を一つだけ書く。

「自社の眠っているアーカイブ素材 × 35〜64歳のニーズ × 帯番組型(フォーマット①)」を、まず1本だけ走らせる。

地方局には、過去10〜30年分の地域映像アーカイブがある。祭、職人、農産物、地場産業、地元の人物、町並みの変遷。これらは、地方の50代以上にとっては「自分の人生の風景」であり、首都圏や海外の35〜64歳にとっては「日本のローカル資源」だ。

週1本、20分、固定MC、固定構成。1クール(12本)走らせて、CPM・登録者増・コメント熱量の3指標で意思決定する。撮り下ろしを最小化し、アーカイブの再編集中心にすれば、1本あたりの制作費は10〜20万円に収まる(推定 / 要検証)。

そして、ここが「日本の良さを海外に経済価値として還流させる」入り口になる。英語字幕(自動 + 軽い人手校正)をつけて、海外の日本好き35〜64歳に届ける。広告CPMは、海外視聴のほうが日本国内より1.5〜2倍高い帯がある(推定 / 業界実勢 / 信頼度:中)。地方資源を、ローカルで磨き、グローバルで換金する。編成発想とROICを両輪で回せば、地方局は "縮む産業" ではなく "走り直せる産業" になる。

結び

YouTubeは新しいテレビではない。テレビという機能が、YouTubeという器に乗り換えただけだ。編成という技術は、まだ死んでいない。むしろ、これからの10年で最も希少な技術になる。

リビングのソファで、35〜64歳がリモコンを握っている。そのリモコンの先に、地上波とYouTubeが並んで並走している。編成を知っている人間が、いま動けば、間に合う。
